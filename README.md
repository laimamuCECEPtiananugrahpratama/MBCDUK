# ğŸŒ‘ MBCDUK: Nightmare Shift

**MBCDUK (Malam Bersama Cecep & Ujangkedu)** adalah game horor psikologis bertema *survival management* dengan estetika **Low-Poly PS1 Retro**. Kamu terjebak dalam shift malam untuk memasak "Ujangkedu" sambil menjaga kewarasan dari gangguan anomali geometri yang mengintai di kegelapan.

## ğŸ•¹ï¸ Cara Bermain (How to Play)

Game ini fokus pada manajemen fokus dan pendengaran.

1. **Masak (Cook):** Tekan tombol **COOK NOW** di HP hingga progress mencapai 100%.
2. **Waspada (Monitoring):** - Gunakan tombol `A` atau `Panah Kiri` untuk mengecek ruangan **Belakang**.
   - Gunakan tombol `D` atau `Panah Kanan` untuk mengecek ruangan **Depan**.
   - Gunakan tombol `S` atau `Panah Bawah` untuk kembali fokus ke HP.
3. **Anomali:**
   - **Cecep (Kotak):** Muncul di Depan. Tanda: Layar bergetar/berat.
   - **Bibin (Bulat):** Muncul di Belakang. Tanda: Lampu berkedip/suara langkah.
   - **Starbyna (Hati):** Menginfeksi HP. Tanda: Layar pink/glitch. **Segera matikan HP (Tombol P)** selama beberapa detik.
